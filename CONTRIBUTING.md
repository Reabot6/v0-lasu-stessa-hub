Contributing to LASU STESSA Resource Hub

Thanks for your interest in contributing!
This project is fully open source, and we welcome improvements of all kinds.

How to Contribute
1. Fork the Repository

Click the Fork button on GitHub and clone your copy:

git clone https://github.com/YOUR_USERNAME/stessa-resource-hub.git
cd stessa-resource-hub
2. Create a New Branch
git checkout -b feature-name
3. Install Dependencies
pnpm install
4. Run the Project
pnpm dev
Types of Contributions Welcome
Bug fixes
UI improvements
Code cleanup
Documentation upgrades
Additional features (new resource filters, analytics, etc.)
Performance optimizations
Supabase schema improvements
Project Standards
Code Style
TypeScript strongly recommended
Follow existing component patterns
Keep functions pure where possible
Use meaningful commit messages
UI
Use Tailwind CSS & shadcn/ui
Ensure responsive design
Avoid inline styles unless necessary
Submitting a Pull Request

Push your changes:

git push origin feature-name
Open a Pull Request on GitHub:
Clear title
Short description of what changed
Screenshots if UI-related
Reference any issues

Your PR will be reviewed promptly.

Reporting Issues

Open an issue on GitHub and include:

Description of the problem
Steps to reproduce
Expected behavior
Screenshots (if UI)
Security Notes

Do NOT commit private keys or real credentials.
Use .env.local and add it to .gitignore.
